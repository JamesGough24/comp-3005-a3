# comp-3005-a3

## Compile & Run Steps
1. Open pgAdmin on your computer
2. Create a database called 'a3_db'
3. Create and insert data into a table called students using 'create_students.sql'
4. Navigate to the project folder in your terminal
5. Compile the JDBC file using the command 'javac src/Main.java'
6. Run the program using the command 'java src/Main'
